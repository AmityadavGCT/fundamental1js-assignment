Q.3 List and describe the five primitive data types in javascript?
Ans: In Javascript, there are five basic, or primitive, types of data. The five most basic types of data are:-
 1. strings:- The string data type in JavaScript represents a sequence of characters that are surrounded by single or double quotes.
 2. numbers:- Number data type in javascript can be used to hold decimal values as well as values without decimals.
 3. booleans:- Represent a logical entity and can have two values: true or false.
 4. undefined:- A variable that has not been assigned a value is undefined.
 5. null:- This type has only one value that is null.
